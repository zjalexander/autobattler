# Next Improvements

1. **Drag/drop polish**
   Add clearer drop targets, invalid-drop feedback, and support dragging bench units into exact board slots reliably across browsers.

2. **Pause combat while editing formation**
   Keep idle combat continuous generally, but add a short formation grace window after buying/moving units so changes feel intentional.

3. **Unit details popup**
   Add an internal window or hover panel showing full stats, traits, ability text, star level, and current trait bonuses.

4. **Combat log clarity**
   Replace terse recap lines with structured attack summaries, round result history, and expandable detail for damage events.

5. **Idle automation upgrades**
   Add prestige-unlocked automation such as auto-reroll, auto-buy wanted units, auto-field upgrades, and auto-buy item slots.

6. **Prestige upgrade tree**
   Let prestige points buy persistent upgrades instead of only flat bonuses: starting gold, shop odds, offline cap, item slot discounts, etc.

7. **Balance spreadsheet expansion**
   Move more formulas and knobs into CSVs, especially enemy scaling, unit abilities, trait breakpoints, item slot costs, and prestige effects.

8. **Save management hardening**
   Add save version migrations, import validation, backup slots, and visible warning/error states for failed imports.

9. **Mobile/touch support**
   Add tap-based fallback controls for drag/drop, tighter internal windows, and responsive board/bench layouts for smaller screens.
